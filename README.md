# Shopping-List

A cool looking dynamic Shopping List

## Prerequisites

None

## Getting Started

On terminal:
1. Run `git clone https://github.com/sgtbrunner/shopping-list.git`
2. Run `cd shopping-list`
3. Open `index.html`

## Built With

* HTML
* CSS
* Javascript
